## 5 查看系统平台Linux/Windows（platform）




```python
import platform
 
def isWindowsSystem():
    return 'Windows' in platform.system()
 
def isLinuxSystem():
    return 'Linux' in platform.system()
 
print isWindowsSystem()
print isLinuxSystem()
```
